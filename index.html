현실에 안주하지 않으며,

공부를 게을리하지 않습니다!

기능 구현을 위해 자바스크립트를 공부하였으며, 몇가지 게임과 바탕화면 시스템을 구현해 보았습니다.

  • 견고한 마크업

    시멘틱 마크업 등을 통해 웹표준성과 웹접근성을 보장하여, 다양한 사용자들이 다양한 기기를 통해 접근할 수 있도록 합니다.

  • UI 개발 능력

    HTML, CSS, JQUERY 라이브러리 등을 이용한 UI 개발 능력이 탁월하며, 더욱 다양하고 동적인 표현을 위해 JAVASCRIPT를 꾸준히 공부하고 있습니다.

  • 반응형 최적화

    모든 홈페이지 프로젝트를 반응형으로 진행해 왔기 때문에, 반응형에 최적화 된 작업 능력을 가지고 있습니다.

  • 협동 능력

    웹디자인 업무를 한 경험으로 디자이너와의 협업이 원할하며, 개발자와도 끊임없는 소통을 통해 최고의 결과물을 얻어냅니다.
